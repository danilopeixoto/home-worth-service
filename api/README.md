# Home Worth API

A home appraisal API.

## Prerequisites

* [OpenJDK (>=11)](https://openjdk.java.net)

## Tests

* Controllers and services are testable.
* Controller tests are unit or integration tests for the REST layer.
* District-related tests not implemented.
